Ohmyfood

Web

Détails du projet

Description

Sur la page principale :
  • La demande était de dynamiser une page web avec des animations CSS suivant le modèle d'une maquette.
  • En version First Mobile et Bureau.
  • J'ai personnalisé le "loader" a la couleur de la société.
  • Principale d'utilisation du CSS :
    • Flexbox
    • Keyframes
    • Animation
    • Transition
    • Z-index
    • Fill-color
  • Le logo a été écrit afin de lui garder une netteté à tout support sans alourdir via une image.
  • Mise en place d'un futur composant "localisation" pour trouver des restaurants proches d’un certain lieu.
  • Création de "mixin" afin de pouvoir réutiliser les codes couleur tels que le bouton d'action, ce qui permet de simplifier le code pour de futures modifications.
  • Utilisation d'icônes via fontawesome
  • Via "::before" j'ai pue y intégrer "counter-increment: compteListe" afin de faire une liste à point automatique.
  • Chaque carte peut être cliquée dans son intégralité, ce qui permet d’atteindre la page des menus.
Sur la seconde page :
  • Un "header" est présent pour chaque société.
  • Une flèche "retour" pour aller à la page d’accueil.
  • À l’arrivée sur la page, les plats apparaissent progressivement avec un léger décalage dans le temps.
  • J’ai utilisé un "::After" pour l’effet de soulignement pour chaque élément des sous-titres
  • Sur chaque survol d'un plat, une petite coche apparaîtra à droite de la ligne. Elle coulisse de droite à gauche. Et si l’intitulé du plat est trop long, ce dernier est rogné avec des points de suspension.
