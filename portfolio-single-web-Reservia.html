Détails du projet

Description
La demande était de créer une page principale du site de réservation suivant le modèle d'une maquette.
En version Desktop et Mobile.

Les liens de la barre de navigation ou plus exactement « Hébergements” et “Activités” sont ancrés pour une navigation plus rapide et pour une meilleure identification.
J’ai également incorporé les couleurs de la marque avec une slide sur le haut de la navigation avec un léger soulèvement

Les usagers pourront rechercher des hébergements dans la ville de leur choix. En ce qui concerne les filtres un changement d’apparence au survol est apparent. J’ai utilisé les icônes via fontawesome
Le champ de recherche est mis avec une écriture noire dès que l’utilisateur écrit pour indiquer la bonne prise en main.
Chaque carte d’hébergement ou activité peut être cliquée dans son intégralité et a une transition pour une meilleure identification.
Le site est également responsive avec des modifications tels que la loupe, menus modifié, la zone populaire passer en premier.
Le test W3C validator est exécuté sur l’ensemble du projet sans erreurs et le site Web est lu par différents navigateurs

Petit Plus :
  • J’ai utilisé Google fonts via le lien direct. (moins lourd lors du chargement)
  • Insertion Favicon
  • J’ai intégré les balises sémantique les plus appropriés afin que les robots puissent mieux identifier le site web ce qui lui permet d’améliorer son référencement.
